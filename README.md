# SaaS Template Project

Командний проєкт: SaaS шаблон вебзастосунку з базовими сторінками, системою авторизації та блогом.

---

## Опис проєкту

Цей проєкт є шаблоном SaaS вебзастосунку, який включає основні сторінки (Index, About, FAQ, Contact)

Проєкт створено для практики командної розробки, структури фронтенду та роботи з GitHub.
